HTML Basics Tutorial

Basic HTML Structure and Syntax

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a webpage. Let's explore the basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Your Page Title</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is a basic HTML page.</p>
</body>
</html>

The <!DOCTYPE html> declaration defines the document type and version of HTML. The <html> element is the root of the HTML document, and the <head> section contains metadata. The <body> section contains the content of the page.

Common HTML Tags and Usage

HTML uses tags to define elements on a webpage. Here are some common tags and their usage:

  • <h1> to <h6>: Headings of different levels.
  • <p>: Paragraph.
  • <a>: Anchor for creating hyperlinks.
  • <img>: Image tag.
  • <ul>, <ol>, <li>: Unordered and ordered lists.
  • <div>: Division for grouping content.
  • <span>: Inline container.
  • <strong> and <em>: Text formatting for strong and emphasized text.
