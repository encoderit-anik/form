Python Basics Tutorial

Introduction to Python Syntax

Python is a high-level, interpreted programming language known for its simplicity and readability. Let's introduce the basic syntax:

# Your first Python program
print("Hello, World!")

The above example prints "Hello, World!" to the console. Python does not require explicit declarations or semicolons.

How to Write Arrays in Python

Arrays in Python are represented using lists. Here are examples and best practices:

# Creating a list (array)
numbers = [1, 2, 3, 4, 5]

# Accessing elements
print(numbers[0]) # Output: 1

# Modifying elements
numbers[1] = 10

# Adding elements
numbers.append(6)

# Removing elements
numbers.remove(3)

# Iterating through the list
for number in numbers:
  print(number)

The above example demonstrates creating a list, accessing and modifying elements, adding and removing elements, and iterating through the list.
