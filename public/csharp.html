C# Basics Tutorial

Basic Syntax of C#

C# is a versatile programming language developed by Microsoft. Let's explore the basic syntax:

using System;

class Program
{
  static void Main()
  {
    // Your code goes here
  }
}

The basic structure includes the using directive to include namespaces and a class named Program with a Main method where the program execution starts.

Functions in C#

Functions in C# are blocks of reusable code. Here's the definition, syntax, and an example:

using System;

class Program
{
  static void Main()
  {
    SayHello("John");
  }

  static void SayHello(string name)
  {
    Console.WriteLine($"Hello, {name}!");
  }
}

The above example defines a simple function SayHello that takes a parameter name and prints a greeting.
