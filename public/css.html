CSS Basics Tutorial

Introduction to CSS and Its Role

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. Its primary role is to enhance the visual presentation and layout of HTML elements. Key points about CSS:

  • CSS separates the structure (HTML) from the presentation (CSS) of a webpage.
  • It allows for consistent styling across multiple pages of a website.
  • CSS follows a cascading order of styles, allowing styles to be inherited and overridden.
  • Selectors are used to target HTML elements for styling.

Basic CSS Properties and How to Use Them

Let's explore some basic CSS properties and how to apply them to enhance the styling of HTML elements:

body {
  font-family: 'Arial', sans-serif;
  margin: 20px;
  background-color: #f8f8f8;
  color: #333;
}

h1 {
  color: #0077cc;
}

p {
  line-height: 1.6;
}

/* Additional Styles */
section {
  margin-bottom: 20px;
  padding: 15px;
  background-color: #fff;
  border: 1px solid #ddd;
  border-radius: 5px;
}

h2 {
  color: #555;
}

footer {
  background-color: #333;
  color: #fff;
  text-align: center;
  padding: 10px;
}

The above CSS code demonstrates the usage of properties such as font-family, margin, background-color, color, and additional styles for a section, heading (h2), and footer. These properties control the font, spacing, colors, and backgrounds, providing a foundation for styling web pages.
